# Emacs Major mode for Rhombus dialect of Racket

I guess someone has to write it....

This library provides a rudimentary GNU+Emacs mode for Rhombus.

It currently provides:
 - Syntax highlighting
 - Very rudimentary indentation?
 - Beginning of defun
